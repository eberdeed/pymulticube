Class Camera

source code

Camera: A camera class that processes input and calculates the corresponding Euler Angles, Vectors and Matrices for moving the camera about a 3-dimensional landscape. For use in OpenGL. This class relies heavily on GLM the OpenGL Math Library.

__init__(self, width, height, position=vec3(0.0,0.0,2.0), focus=vec3(0.0,0.0,0.0))
(Constructor)

source code 

Constructor with vectors to define viewer position and focus. width : the width of the viewport. height : the height of the viewport. position : the position of the camera. focus : the position of the object looked at.

setCamera(self, width, height, posX, posY, posZ, upX, upY, upZ, yaw, pitch)

source code 

Set the camera values to define viewer postion and Euler angles for direction. width : the width of the viewport. height : the height of the viewport. posX, posY, posZ : the (x, y, z) position of the camera. upX, upY, upZ : the (x, y, z) up direction of camera up. yaw : yaw based on (x > 0, 0, 0) as zero radians (right <-> left). pitch : pitch based on (x, 0, z) as zero radians (up <-> down).

processKeyboard(self, direction, deltaTime)

source code 

Processes input received from any keyboard-like input system. Accepts input parameter in the form of camera defined index, such as Camera.Camera_Movement.index("RIGHT") or Camera.Camera_Movement.index("UP") (to abstract it from windowing systems). This is adjustable using the SPEED variable.

processMouseMovement(self, xoffset, yoffset)

source code 

Processes input received from a mouse input system. It expects the offset value in both the x and y direction. This is adjustable using the SENSITIVITY variable.

processMouseScroll(self, inout)

source code 

Process input received from a mouse scroll-wheel event. This method only requires input on the wheel-axis
